# 2026 男足世界杯观赛指南小程序页面结构与低保真原型

## 1. 页面结构总览

```mermaid
flowchart TD
  A["首页：今晚赛事"] --> D["比赛详情页"]
  B["赛程"] --> D
  C["赛果"] --> D
  E["球队"] --> F["球队详情页"]
  F --> D
  G["场馆"] --> H["场馆详情页"]
  H --> D

  A -.底部 Tab.-> B
  B -.底部 Tab.-> C
  C -.底部 Tab.-> E
  E -.底部 Tab.-> G
```

## 2. 底部导航

第一版建议使用 5 个 Tab：

1. 首页
2. 赛程
3. 赛果
4. 球队
5. 场馆

比赛详情、球队详情、场馆详情作为二级页面，不放入底部导航。
